# Bank Account Assignment

This folder contains the implementation of the Bank Account assignment for the Python OOP module.

## Files
- `bank_account.py`: Contains the `BankAccount` class which handles basic banking operations like deposits, withdrawals, and interest calculation.
- `user_with_bank_accounts.py`: Demonstrates the association between a `User` class and the `BankAccount` class, allowing users to manage multiple accounts.

## Key Features
- **Object Oriented Programming**: Use of classes and objects to represent bank accounts and users.
- **Chaining**: Methods are designed to return `self`, allowing for method chaining.
- **Class Association**: The `User` class is associated with `BankAccount` instances through a dictionary.

## How to Run
You can run the scripts using Python:
```bash
python bank_account.py
python user_with_bank_accounts.py
```
