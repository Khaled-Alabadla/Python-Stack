# 🥷 Ninja Gold

A fun mini-game built with Flask where you help a ninja gather gold by visiting various locations!

## 🎮 Game Description
The goal is to gather **500 gold** in less than **15 moves**. You can visit four different places, each offering different risks and rewards:
- **Farm**: Earns 10-20 gold.
- **Cave**: Earns 5-10 gold.
- **House**: Earns 2-5 gold.
- **Casino**: Earn or lose up to 50 gold!

## ✨ Features
- **Dynamic Activity Log**: Tracks every move with timestamps and color-coding.
- **Win/Loss States**: Special screens for when you achieve your goal or run out of moves.
- **Premium UI**: Modern dark-themed interface with smooth transitions and responsive design.

## 🚀 Getting Started
1. Install Flask:
   ```bash
   pip install Flask
   ```
2. Run the application:
   ```bash
   python server.py
   ```
3. Open your browser and navigate to `http://localhost:5000`
