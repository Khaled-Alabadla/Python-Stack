# Dojo Fruit Store

A Flask web application for ordering fruits.

## Features

- Order fruits (Strawberry, Raspberry, Apple, Blackberry) with quantities
- View all available fruits with images
- Checkout and see order summary

## How to Run

1. Ensure you have Python and Flask installed.
2. Run `python server.py`
3. Open your browser to `http://localhost:5000`

## Routes

- `/`: Home page with order form
- `/fruits`: Display fruit images
- `/checkout`: Process order (POST)
