# HTML Table Flask Application

## Description

A Flask web application that displays a list of users in an HTML table format.

## Project Structure

```
Flask/HTML_Table/
├── server.py           # Main Flask application
├── templates/
│   └── index.html      # HTML template with user table
└── README.md           # Documentation file
```

## Features

- Displays user data (first name, last name, and full name) in a responsive table
- Uses Bootstrap 5 for styling
- Built with Flask framework

## Usage

1. Install Flask: `pip install flask`
2. Run the server: `python server.py`
3. Open your browser and navigate to `http://localhost:5000/`
