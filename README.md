## 🚀 Featured Project: Zoo Management System

I recently completed the **Zoo Management System**, a hands-on project exploring OOP principles:
- **Encapsulation:** Managed animal state within specialized classes.
- **Inheritance:** Derived specific species from a base `Animal` class.
- **Polymorphism:** Overrode behaviors like feeding and info display for different animals.

You can find the detailed implementation and steps in the **[OOP README](./OOP/zoo/README.md)**.
